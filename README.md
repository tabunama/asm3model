# asm3model

Python package for ASM3 compiled extension modules.

This repository is intended to host the project documentation, release metadata, and license / attribution files for the `asm3model` package.

## Installation

Install from PyPI:

```bash
pip install asm3model
```

## Import

```python
from asm3model import asm3, combiner, carboncombiner, hyddelay, settler1d
```

## Current modules

- `asm3`
- `combiner`
- `carboncombiner`
- `hyddelay`
- `settler1d`

## Repository contents

This repository is prepared to keep:

- `README.md`
- `LICENSE`
- `THIRD_PARTY_NOTICES.md`
- `docs/`

Build outputs, local development artifacts, generated binaries, and local-only source/build helpers are intentionally not tracked here.
