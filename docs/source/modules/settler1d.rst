One-dimensional settler module
==============================

Import
------

.. code-block:: python

   from asm3model import settler1d

Class
-----

``Settler1D(SETTLERINIT, SETTLERPAR, DIM, LAYER, TEMPMODEL, ACTIVATE, PAR_ASM3, SETTLER)``

Parameters
----------

- ``SETTLERINIT``: initial settler state vector
- ``SETTLERPAR``: settler parameter vector
- ``DIM``: settler dimensions
- ``LAYER``: settler layer definition
- ``TEMPMODEL``: temperature model flag
- ``ACTIVATE``: dummy-state activation flag
- ``PAR_ASM3``: ASM3 parameter vector
- ``SETTLER``: reactive-settler flag

Notes
-----

This page documents the module-level interface intended for future flowsheet construction.
